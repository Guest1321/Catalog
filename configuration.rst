Server Requirements :
^^^^^^^^^^^^^^^^^^^^^
1. PHP 7.4.3 or higher
2. mysql 15.1 or higher
3. Apache 2.4.41
4. htaccess / rewrite engine

^^^^^^^^^^^^^
Configuration
^^^^^^^^^^^^^^
1. Buka file ./application/config/config.php , pada line 26 ubah 'http://localhost/catalog/' menjadi 'linkdomainkamu' (misal https://catalog-online.com atau http://localhost/`catalog`_). Jika ditaruh di local server maka teks kuning sesuaikan dengan nama folder.
2. Untuk setting database sillahkan buka di ./application/config/database.php ubah pada line 78 - 81.
